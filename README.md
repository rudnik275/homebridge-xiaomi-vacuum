# Pre Requirements
> nodejs.version >= 12

Package must contain `secret.json` file
with `ip` and `token`.
 
[Instruction](https://python-miio.readthedocs.io/en/latest/discovery.html#apple) to get token
```json
{
  "ip": "xxx.xxx.xxx.xxx",
  "token": "********************************"
}
```

# Getting started
- `npm i`
- create `secret.json`
- `npm run start`
- scan QR code from terminal in your "Home" app
